Markdown语法

Markdown语法不难，但是内容还算挺多，自己多用用就熟悉了…

1.标题

1.1 使用#,可表示1-6级标题

效果略

1.2 使用=和-标记一级和二级标题

一级标题
=========
二级标题
---------

一级标题

二级标题


2. 列表

使呈现列表状

无序列表

* 无序列表
+ 无序列表
- 无序列表

有序列表

1. 2. 3. 有序列表

效果：

  • 无序列表(*)
  • 无序列表(+)
  • 无序列表(-)
  1. 有序列表
  2. 有序列表
  3. 有序列表


3. 分割线

分割线

分割线最常使用就是三个或以上*还可以使用-和_


4.强调

在强调内容两侧分别加上*或者_如：

*斜体*，_斜体_
**粗体**，__粗体__

效果：

斜体，_斜体_
粗体，粗体


5.区块引用

在段落的每行或者只在第一行使用符号>,还可使用多个嵌套引用,如：

>区块引用
>>嵌套引用

效果：

区块引用

嵌套引用


6.代码区块

代码区块的建立是在每行加上4个空格或者一个制表符（如同写代码一样）。如普通段落：

printf(“Hello, Markdown.”);

效果：


7. 链接

链接可以由两种形式生成：行内式和引用式

行内式：

[好看的姑娘](https://tinger-mefus.github.io/assets/Markdown.png/)
　好看的姑娘

引用式：

[好看的姑娘][1]
[1]: https://tinger-mefus.github.io/assets/Markdown.png/
　好看的姑娘

注意：[1]: https://tinger-mefus.github.io/assets/Markdown.png/这一句不能出现在区块中


8.图片

添加图片的形式和链接相似,只需在链接的基础上前方加一个！

![文字](链接)

9.反斜杠 \

相当于反转义作用,使符号成为普通符号

\

例如：

[1. 苹果]是有序列表

  1. 苹果

[1\. 桃子] 不是有序列表

1. 桃子
